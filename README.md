# Django Todo App

This is a simple Todo App built using Django.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed

## Technologies Used
- Python
- Django
- HTML
- CSS

## How to Run Project

1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Run server

### Commands

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Open browser:
http://127.0.0.1:8000/
✅ 2️⃣ requirements.txt
Create a file named:
requirements.txt
Add this inside:
Django>=4.0,<5.0
If you want exact version, run in terminal:
pip freeze
Copy Django version from there.
